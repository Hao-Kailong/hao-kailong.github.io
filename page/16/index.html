更新于 2020-08-14 | 分类于 Neural Networks | 评论数：

1. SGD

Batch Gradient Descent

在每一轮的训练过程中，Batch Gradient Descent算法用整个训练集的数据计算cost function的梯度，并用该梯度对模型参数进行更新：

优点:

  • cost fuction若为凸函数，能够保证收敛到全局最优值；若为非凸函数，能够收敛到局部最优值

缺点:

  • 由于每轮迭代都需要在整个数据集上计算一次，所以批量梯度下降可能非常慢
  • 训练数较多时，需要较大内存
  • 批量梯度下降不允许在线更新模型，例如新增实例。

Normalization神经网络中的正则化

更新于 2020-04-20 | 分类于 Neural Networks | 评论数：

在神经网络中经常需要用到正则化方法，来加速神经网络的训练以及减少震荡等

Batch Normalization

对每个batch中同一维度的特征做normalization

Layer Normalization

针对单个训练样本的所有维度做normalization。之前的BN是纵向规范化，而layer normalization是横向规范化

Instance Normalization

与BN只有一点不同，BN是作用于一个batch，而IN是作用于单个样本

Group Normalization

划分区域，然后进行normalization
