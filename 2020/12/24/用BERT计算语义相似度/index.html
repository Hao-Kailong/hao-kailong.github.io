用BERT计算语义相似度

更新于 2020-12-27 | 分类于 NLP ， Pretrained Language Model | 评论数：

最近在实验中需要计算句子之间的相似度，偶然看到了一篇文章，有以下说法：

自2018年BERT惊艳众人之后，基于预训练模型对下游任务进行微调已成为炼丹的标配。然而近两年的研究却发现，没有经过微调，直接由BERT得到的句子表示在语义文本相似性方面明显薄弱，甚至会弱于GloVe得到的表示。此篇论文中首先从理论上探索了masked language model 跟语义相似性任务上的联系，并通过实验分析了BERT的句子表示，最后提出了BERT-Flow来解决上述问题。

BUG!

在使用torch.max()的过程中，发现了官方文档的一个bug，在官方文档中，这样写道：

https://pytorch.org/docs/stable/generated/torch.max.html?highlight=torch%20max#torch.max

If there are multiple maximal values in a reduced row then the indices of the first maximal value are returned.

即，如果存在多个最大值，返回第一个最大值对应的下标。
